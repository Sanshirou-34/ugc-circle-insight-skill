# UGC 圈层洞察 Skill

帮助品牌与设计团队从当前 UGC 内容、评论和互动行为中，快速理解陌生圈层的概念、情绪、黑话、互动习惯与活动机会。

它不会停留在术语释义，而会从表层词汇继续下钻到核心实践、母文化、身份群体和关联概念。例如研究「痛屋」时，会同时追溯痛文化、二次元群体，以及痛包、痛车、痛娃等概念家族。

## 适用场景

- 快速进入陌生圈层或亚文化
- 为 UGC 活动寻找机制与参与动机
- 提炼圈内黑话、情绪张力和互动习惯
- 为视觉包装与品牌沟通提供依据

## 使用方式

在 Codex 中发送：

```text
请从 https://github.com/Sanshirou-34/ugc-circle-insight-skill/tree/main/skills/ugc-circle-insight 安装 ugc-circle-insight skill
```

安装后可以这样提问：

```text
使用 ugc-circle-insight 调研「痛屋」。
```

Skill 会先确认小红书、抖音等 UGC 平台的登录与授权状态，再开始研究；若平台无法访问，会明确标记证据缺口并降级使用公开网页信息。

## 研究节奏

默认先给出一版可快速阅读的圈层速览，再询问：

> 你是想做活动机制、视觉包装，还是品牌沟通？

用户选择方向后，再继续深挖。输出同时覆盖洞察和建议，但以活动玩法为主。

## 研究方法预览

完整版本见 [`research-method.md`](skills/ugc-circle-insight/references/research-method.md)。核心研究流程如下：

1. **隔离研究对象**：为每次研究单独建立证据账本，以 `S01`、`S02` 编号，避免不同圈层或历史记忆相互污染。
2. **确认范围与平台**：选择圈层真实发布和互动的 2–3 个平台；默认观察近 90 天，原生 UGC 内容与评论优先。
3. **建立概念地图**：先判断用户输入的是物件、行为、场景、审美、梗，还是真正的圈层名称，再向母文化和身份群体下钻。
4. **向外搜索、反向验证**：扩展别称、行为、角色、同族概念和情绪张力，同时寻找误用、争议、跟风与圈外采用等反证。
5. **记录证据强度**：区分直证、同族、母文化、邻接与反证，并将信号标记为强、中、弱。
6. **形成证据链**：从可见内容与互动出发推断情绪，再转化为设计启示和参与风险。
7. **通过追问深挖**：速览后围绕目标人群、底层需求、参与行为和品牌角色继续研究。

研究不会只匹配用户给出的关键词，而是使用下面的结构寻找更底层的圈层关系：

```text
表层术语 → 重复实践 → 母文化 → 身份群体 → 情绪 / 社会功能
```

例如研究「痛屋」时，不只搜索痛屋，还会验证它与痛文化、二次元群体、痛包、痛车、痛娃等概念之间的关系，并区分核心、同族、邻接和跨圈使用。

每条重要洞察都需要形成证据链：

```text
内容信号 → 可见互动 → 可能情绪 → 替代解释 → 设计启示 → 参与风险
```

证据账本中的每个来源都会带有标签：

```text
S01 [平台] [原帖|评论|摘要|转载|媒体]
    [直证|同族|母文化|邻接|反证]
    [概念层级] [样本角色] [日期] [强|中|弱]
```

## 报告模板预览

完整版本见 [`report-template.md`](skills/ugc-circle-insight/references/report-template.md)。速览报告默认包含：

### 1. 一句话圈内翻译

用圈内人能够认可、圈外人能够理解的方式解释术语，同时说明它不是什么，以及它属于主流、新兴、小众、争议、过时还是待验证用法。

### 2. 圈层地图与概念星系

```text
表层术语 → 核心实践 → 母文化 → 身份群体 → 情绪 / 社会功能
```

| 概念或层级 | 关系 | 共享机制 | 特有变量 | 信号 | 证据 |
|---|---|---|---|---|---|
| 目标术语 | 核心 / 母文化 / 同族等 | 跨概念不变量 | 目标特有差异 | 强 / 中 / 弱 | S01、S02… |

### 3. 当前脉搏

列出 3–5 个当前模式，分别说明信号强度、来源、可见事实和当下意义，并区分长期文化与短期热点。

### 4. 语言、符号与互动

| 信号 | 圈内用法或行为 | 可能情绪 | 替代解释 | 误用风险 | 证据 |
|---|---|---|---|---|---|

### 5. 圈层张力

指出最重要的 1–3 个圈内边界、矛盾或争议，以及它们如何影响参与意愿。

### 6. UGC 活动机会

每个活动方向都会包含参与动作、情绪杠杆、传播或共创钩子、品牌角色、视觉语气、翻车风险和小规模验证方式。

### 7. 证据、局限与策略追问

报告会公开研究日期、平台访问状态、样本数量、来源链接与降级方案，并根据证据提出 1–2 个真正影响策略选择的问题，最后询问：

> 你想深挖活动机制、视觉包装，还是品牌沟通？

## 数据边界

只使用当前可访问的公开内容或用户已授权访问的页面，不绕过登录、验证码、访问限制或平台安全机制。研究结论会按来源类型标记，并区分事实、观察、推断和建议，避免把单条内容误判为圈层共识。
